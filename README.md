# Automated E-Commerce Sales Dashboard & PDF Report Generator

This project automates e-commerce sales analysis using Python, Pandas, and Matplotlib.
The system processes raw sales data, calculates business KPIs, generates professional charts, creates Excel dashboards, 
and automatically exports PDF business reports.
It is designed for business reporting, sales analytics, and workflow automation scenarios.


# Features

- Load and process e-commerce sales datasets
- Generate automated KPI summaries
- Calculate revenue, profit, orders, and profit margins
- Analyze product and category performance
- Create automated business charts
- Export professional PDF reports
- Generate Excel dashboard files automatically
- Business-focused reporting workflow
- Clean and modular Python structure


# KPIs Included

- Total Revenue
- Total Profit
- Total Orders
- Total Units Sold
- Average Order Value
- Profit Margin
- Best Performing Category
- Best Selling Product
- Best Sales Channel


# Technologies Used

- Python
- Pandas
- Matplotlib
- OpenPyXL
- FPDF / PdfPages
- Excel (xlsx)
